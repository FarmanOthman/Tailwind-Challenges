## 🚀 Getting Started

1. **Clone the repository**
   ```bash
   git clone <repo-url>
   cd tailwind-challenges
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Start development server**
   ```bash
   npm run dev
   ```

## 📚 Project Structure

Each challenge follows a consistent structure:
```
challenge-XX/
├── src/

│   ├── assets/        # Images and icons
│   └── App.jsx        # Main application

```
